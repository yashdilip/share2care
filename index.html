Share2Care (Let's make event success)

This will be a web-based project which allows the user to borrows resources from his/her circle to carry out an event successfully. A user will have circle having friends and he/she can request for resources from event-sub-circle friends to organize an event. So that user wonâ€™t have to buy things by own for few days or for a few hour event.

Description:

  • Actor: user, system
    A user will be an actor who will perform the following use-cases
  • Signup: A user can sign up to the system using online portal.
  • Login: An already logged in user can log into the system.
  • Create Circle: A user can create a circle.
  • Send Request: An already registered user can send a request to other registered users asking to join into his/her circle.
  • View: A user can view his/her circle users. A user can view his/her circle/s.
  • Join Circle: A user can join a circle. A user can join multiple circles.
  • Leave Circle: A user can leave the circle.
  • Search: A registered user can search into the system for other registered users.
  • Create Event: A user can create an event and include selected user from his/her circle or all users from his/her circle. The event will have status, list of the participant.
  • Edit Event: An owner user can edit the event. Notification will be sent to included users about changes.
  • Delete Event: A user with owner permission can delete already logged event from the system. The system will notify the users about delete event.
  • Event notifies: The system will automatically notify the users who are included in the event. Optional to notify all the users of the circle.
  • Ask Resource/s: A user can ask for resources/things from the selected users /or all users. The resource will have priority in the event as the user feels about the requirement on that event.
  • Offer Resource/s: A user can offer thing/s notifying to the event owner and all other included users.
